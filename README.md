# OmniChat - Omni-Channel Communication Platform

A modern, ChatGPT-inspired interface for business communication, built with Next.js and Tailwind CSS. OmniChat helps with employee management, recruitment, accounting, and e-invoicing.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```
